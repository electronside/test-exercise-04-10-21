1. Перейдите в папку html и выведите список всех папок (без файлов)
	
	флаг -d только директории

	cd var/www/html
	ls -d
	

2. Удалите пустые строки из файла error.txt

	cd var/www/html/logs
	grep . error.txt
	
	или с помощью встроенного текстового редактора

	nano var/www/html/logs/error.txt

	клавиатурой удаляем сохраняем 
	
3. Выведите список файлов с расширением .jpg в папке images

	find var/www/html/images -name *.jpg 

4. Напишите bash скрипт который переименует все файлы в папке logs с .txt на .log

	for file in *.txt; do
    	mv "$file" "$(basename "$file" .txt).log"
	done

	так несколько расширений

	for fname in *.{html,htm}
	do
   	mv -v "$fname" "${fname%.}.xml"
	done
	
	или

	find . -iname *.txt  -exec mv {} "$(basename {} .txt).log" \;

5. Модифицируйте права доступа и владельца папки images следующим образом: Пользователь www-data и группа www-data имеют чтение/запись на всю директорию и на файлы внутри директории остальные не имеют доступа к папке и файлам.

	cd var/www/html/images
	ls -l images	отображает права доступа владельца и группы

	-rwxr–rw- 1 antone antone 0 Sep  6 14:59  Images

	-rwxr-rw 	показывает права доступа
	1		дополнительное имя
	antone	 	владельца и его группу
	0		размер папки
	Sep  6 14:59	дата последнего изменения
	Images		имя папки

	rwx — oзначает чтение запись выполнение и имеет свой собственный номер
	read чтение– 4
	write запись – 2
	x выполнение – 1
	
	4 + 2 + 1 = 7 все права доступа для чтения записи и выполнения

	7 - владелец
	7 - группа
	7 - другие

	Команда change mod
	если команда change mod 777 var/www/html/images
	установит чтение запись для всех типов пользователей

	тогда chmod 770 ./var/www/html/images
	ограничит чтение запись для других кроме владельца и группы
	
	В зависимости от файловой и операционных системы права бывают разные.
	Права на фыйловую систему в Windows проще она не поддерживает отдельное 
	право на выполнение файлов то есть что если файл можно читать то и выполнять 
	для пользователя в отличии от Unix систем где права различны 
	

6. Удалите все содержимое в файле access.txt не удаляя сам файл.
		
	truncate --size 0 var/www/html/logs/access.txt

7. Выведите список файлов в папке images размер которых более 2 мегабайт
	
	если в ней только картинки тогда может

	find var/www/html/images . -type f -size +2M -ls
